<h3>Description</h3>
Displays random non-alcoholic drinks
<h3>Requirements</h3>
.Net 5.0 & NodeJs

<h3>To run backend:</h3>
$ cd backend <br />
$ dotnet restore <br />
$ dotnet run <br />

<h3>To run frontend: </h3>
$ cd frontend<br />
$ npm i <br />
$ npm run <br />

Source api : https://www.thecocktaildb.com/api.php
